📌 About the Workshop

On Wednesday, Mlab in partnership with GIZ hosted a hands‑on session for the Stepup II cohort in Tshwane. We applied the full product development lifecycle — from stakeholder analysis to deployment — using real‑world user data from LetterBoxd, a social platform for film lovers with over 18 million users.

Phase 1–2

Planning & Analysis

Acted as BAs and UX/DBA team: interviewed stakeholders, defined segmentation (subscription tier), analysed user habits.

Phase 3–4

Design & Implementation

Wireframes → high‑fidelity prototype → coded HTML/CSS/JS personas → GitHub Pages deployment.

Phase 5–7

Test, Deploy, Maintain

Tested persona switching, responsiveness, data accuracy. Deployed publicly. Ready for database integration.

🎓 Key Takeaways from the Lecture

  • LetterBoxd works as a “digital diary” for movies: find, rate/review, save to lists, interact socially, analyse stats.
  • Watchlist size + social connections + session length strongly predict willingness to pay.
  • Free users who only browse (search, home page, trailers) have high churn risk.
  • Patron power users create thousands of lists and have near‑zero churn.
