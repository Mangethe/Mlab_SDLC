# 🗺 User Story Mapping – LetterBoxd Persona Explorer

## Epic: *Enable film lovers to track, share, and discover movies based on their engagement level*

| User Role | Activity | Story | Priority |
|-----------|----------|-------|----------|
| **Free user** (Casual Casey) | Discover trending movies | As a free user, I want to see trending and popular movies on the homepage so I can decide what to watch without effort. | Must have |
| | Quick rating | As a free user, I want to give a star rating without writing a review so I can log movies faster. | Must have |
| | Watchlist basics | As a free user, I want to add movies to a watchlist so I don't forget them. | Must have |
| | Avoid commitment | As a free user, I want to hide review writing prompts so I don't feel pressured. | Could have |
| **Pro user** (Dedicated Dana) | Yearly stats | As a Pro user, I want to see my Year in Review with charts (most watched director, genre, country) so I can share it on social media. | Must have |
| | Review with formatting | As a Pro user, I want to bold/italic text and add emojis in my reviews. | Should have |
| | Filter activity feed | As a Pro user, I want to filter my feed by "friends only" or "critics I follow". | Should have |
| | Export diary | As a Pro user, I want to export my movie diary to CSV for personal analytics. | Could have |
| **Patron user** (Curator Chris) | Unlimited lists | As a Patron user, I want to create unlimited custom lists (e.g., “Every film from the 1940s noir cycle”) with up to 10,000 items per list. | Must have |
| | Backdrops & themes | As a Patron user, I want to customize my profile with movie backdrops and a personal theme. | Should have |
| | Advanced search | As a Patron user, I want to search by actor, director, country, year, AND my own tags. | Must have |
| | Community highlights | As a Patron user, I want to pin my best reviews to the top of my profile. | Could have |

## 🧩 User Journey Map (Simplified)

**Free → Pro → Patron** progression triggers:

1. **Awareness** – User sees a friend's "Year in Review" on Twitter.
2. **Trial** – User signs up (free). Uses watchlist and ratings.
3. **Engagement** – User writes first review, gets a like from a stranger.
4. **Conversion to Pro** – User wants stats and filtering (after 3+ months).
5. **Power use** – User creates 10+ lists, follows 500+ people.
6. **Conversion to Patron** – User hits list limit (free: 3 lists, Pro: 20 lists, Patron: unlimited).
